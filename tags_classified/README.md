# danbooru tag分类数据集

本仓库以 Danbooru2024 及 Danbooru2024 Wiki 为数据源，使用 DeepSeek-V3.1 对图像标签进行分类。

## 分类体系

分为以下 **7 个大类**：

1. **人物本身的特征** - 发色、瞳色、身材、体型等生理特征  
2. **表情** - 面部表情、情绪表现  
3. **服饰** - 衣服、鞋子、配饰、装扮等  
4. **动作** - 身体动作、姿态、行为  
5. **环境/背景** - 场景、地点、环境元素  
6. **物品** - 道具、工具、物品  
7. **构图** - 镜头角度、画面构图、视角

## 数据格式

- 数据保存为 **CSV 格式**
- 每行一个 tag
- 按照 **字母顺序** 排序
